### Task 1: Create ArrowLink primitive

**Files:** `src/components/ui/ArrowLink.astro` (new)

**Action:**
Create an Astro component with this API:

```ts
interface Props {
  href?: string;        // required when decorative=false
  direction?: 'left' | 'right';   // default 'right'
  decorative?: boolean;  // default false
  class?: string;        // optional extra classes (forwarded)
  // Plus rest-spread of any HTML attribute (aria-hidden, target, rel, etc.)
}
```

Render shape:
- When `decorative=false`: `<a class="group inline-block text-base text-secondary hover:text-accent transition-colors focus-visible:..." href={href}>← <span class="underline-classes"><slot/></span></a>` (or `<slot/> →` for right direction)
- When `decorative=true`: `<span class="inline-block text-base text-secondary group-hover:text-accent transition-colors">{...same inner structure...}</span>`

Underline classes (verbatim from existing pattern):
`relative after:absolute after:bg-current after:bottom-0 after:left-0 after:h-px after:w-full after:origin-bottom-right after:scale-x-0 group-hover:after:origin-bottom-left group-hover:after:scale-x-100 after:transition-transform after:ease-in-out after:duration-300`

Arrow placement: text-of-slot wrapped in the underline span; arrow placed OUTSIDE that span (left direction = arrow before, right direction = arrow after).

Pass-through: rest-spread `{...rest}` onto the rendered element so consumers can attach `aria-hidden`, `target`, `rel`, etc.
