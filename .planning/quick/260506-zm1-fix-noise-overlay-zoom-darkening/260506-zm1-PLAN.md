<objective>
Fix the dark rectangular regions that appear when the user zooms in and scrolls. The artifact is caused by `body::after` using `position: fixed`, which is sized to the layout viewport — when zoomed, scrolling reveals regions outside that viewport rectangle that never receive the +3.5% noise lightening, appearing darker by contrast.

Switch the pseudo-element to `position: absolute` and make body its containing block (`position: relative`). The noise then sizes to body's content box (full document height), scrolls naturally with content, and is unaffected by browser zoom.

Output: Uniform background tone at all zoom levels and scroll positions; noise still renders above content.
</objective>
