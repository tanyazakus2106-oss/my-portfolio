  <action>
In the `.container` rule (~lines 218-233), replace the two `padding-left`/`padding-right`
declarations and DELETE the entire `@media (min-width: 768px) { .container { ... } }` block
that overrides them.

Set both padding-left and padding-right to:

    clamp(24px, 4.28px + 5.26vw, 80px)

Math (linear anchors 24px @375px viewport, 80px @1440px viewport):
  slope     = (80 - 24) / (1440 - 375) = 56 / 1065 = 0.05258 = 5.26vw
  intercept = 24 - 0.05258 * 375 = 4.28px
Verification of anchors:
  @375px:  4.28 + 5.26*3.75  = 24.0px  (lower clamp)
  @1440px: 4.28 + 5.26*14.40 = 80.0px  (upper clamp)
Above 1440px the upper bound holds at 80px, which is correct since the container is
centered at max-width:1440px.

Add a brief inline comment noting the fluid range, e.g.:
  /* fluid gutter: 24px @375px viewport -> 80px @1440px */

Do not touch any other rule. Leave the `var(--spacing-lg)` / `var(--spacing-xl)` tokens
defined under @theme intact — only the .container usage changes.
