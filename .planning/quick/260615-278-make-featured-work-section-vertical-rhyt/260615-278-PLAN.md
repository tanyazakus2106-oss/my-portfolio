# Quick Task 260615-278: Fluid vertical rhythm (featured-work section)

## Objective

Make the home page featured-work ("Projects") section's vertical spacing scale
fluidly with viewport width instead of snapping at the `md` (768px) breakpoint —
mirroring the horizontal-gutter treatment from quick task 260615-1js.

Three gaps, all previously `var(--spacing-3xl)` (64px) → `132px` at `md`:

1. Divider → "Projects" heading (section `padding-top`)
2. Between project cards (flex `gap`)
3. Last card → footer (section `padding-bottom`)

## Decision

Scale with **viewport width** (vw), consistent with the horizontal gutter clamp.

Introduce one shared `@theme` token rather than repeating a raw `clamp()` literal
three times in the component (keeps the component free of magic numbers — CLAUDE.md
decision-tree rule #5).

Fluid value (linear anchors 64px @375px viewport, 132px @1440px viewport):
slope = (132−64)/(1440−375) = 68/1065 ≈ 6.39vw; intercept ≈ 40px.

```css
--spacing-section: clamp(64px, 40px + 6.39vw, 132px);
```

## Tasks

### Task 1 — Add fluid section token + apply to featured-work section

- **files:** `src/styles/global.css`, `src/pages/index.astro`
- **action:**
  - `global.css` @theme: add `--spacing-section: clamp(64px, 40px + 6.39vw, 132px);`
    after `--spacing-4xl`.
  - `index.astro` projects `<section>`: replace
    `pt-[var(--spacing-3xl)] md:pt-[132px] pb-[var(--spacing-3xl)] md:pb-[132px]`
    with `pt-[var(--spacing-section)] pb-[var(--spacing-section)]`.
  - `index.astro` cards wrapper: replace
    `gap-[var(--spacing-3xl)] md:gap-[132px]` with `gap-[var(--spacing-section)]`.
- **verify:** `npm run typecheck` (0 errors); prettier clean on the two files.
- **done:** all three gaps scale continuously 64px→132px with viewport width,
  capping at 132px above 1440px; no breakpoint snap.
