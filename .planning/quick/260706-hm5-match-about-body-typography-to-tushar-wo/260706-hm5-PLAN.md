# Quick Task 260706-hm5 — Match About body typography to the reference

## Goal

Bring the About page body copy in line with the design reference
(tushar.work/about), which uses Tailwind's `text-lg` + `leading-relaxed` for
both the intro lede and the section body paragraphs. Replace the About page's
fixed-px values so the body text uses the shared, rem-based type scale.

## Reference (verified from live site, 2026-07-06)

- tushar.work/about intro lede AND "How I Work"/"Beyond" bodies both use:
  `text-lg text-muted-foreground leading-relaxed` (on the wrapper div).
- Stock Tailwind v4 scale: `--text-lg: 1.125rem`, `--leading-relaxed: 1.625`.
- Root font-size bumps `16px -> 17px` at `min-width: 1200px` (same as this repo's
  global.css), so `text-lg` renders 18px below 1200px and ~19.125px at/above it.

## Tasks

### Task 1 — Swap About body copy to text-lg / leading-relaxed

**Files:** `src/pages/about.astro`

**Action:**
- Intro lede `<p>`: `text-[20px] leading-[30px]` -> `text-lg leading-relaxed`
  (kept `mt-[var(--spacing-lg)]`, `text-[var(--color-text-secondary)]`,
  `max-w-[600px]`).
- The four "How I work" / "Beyond work" body `<p>`:
  `text-[18px] leading-[1.625]` -> `text-lg leading-relaxed` (kept muted color).

**Left untouched (out of scope):** section headings
(`text-[28px] md:text-[32px]` — intentionally larger than the reference's
`text-2xl md:text-3xl`), hero grid, image `sizes`, `max-w-[600px]` intro cap,
all spacing.

**Verify:** no `text-[20px]`/`text-[18px]`/`leading-[30px]`/`leading-[1.625]`
remain; 5 `text-lg leading-relaxed` occurrences (1 intro + 4 body); headings
still `text-[28px] md:text-[32px]` (x2).

**Done:** About body copy uses the rem-based scale matching the reference.

## Effect

- Intro: 20px -> 18px (fixed) becomes `text-lg` (18px, ~19.125px >=1200px);
  line-height fixed 30px -> ratio 1.625.
- Section bodies: visually unchanged below 1200px (18px, lh 1.625) but now
  rem-scale to ~19.125px on desktop, matching the reference's behavior.
