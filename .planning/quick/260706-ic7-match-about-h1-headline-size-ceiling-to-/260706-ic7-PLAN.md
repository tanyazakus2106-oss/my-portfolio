# Quick Task 260706-ic7 — Match About H1 size ceiling to reference

## Goal

Bring the About page hero headline down to the same desktop size as
tushar.work/about ("Designing with craft, building with code."), while keeping
the site's fluid `clamp()` headline convention.

## Reference (verified from live site, 2026-07-06)

- tushar H1: `font-serif text-4xl md:text-5xl lg:text-6xl leading-[1.1]`.
- Stock Tailwind v4: text-4xl 2.25rem / text-5xl 3rem / text-6xl 3.75rem.
- Desktop (lg+) = text-6xl = 3.75rem = 60px (≈63.75px at the 17px root ≥1200px).

## Task

**Files:** `src/pages/about.astro` (h1, line 47)

**Action:** two edits on the h1 class only:
1. clamp MAX `4.5rem` -> `3.75rem`:
   `text-[clamp(2.5rem,5vw,4.5rem)]` -> `text-[clamp(2.5rem,5vw,3.75rem)]`
   (caps at tushar's 3.75rem/60px instead of 72px).
2. line-height `leading-[1.05]` -> `leading-[1.1]` (match tushar).

Kept `mt-[var(--spacing-md)]`, `text-[var(--color-text-primary)]`, the min
(2.5rem), and the 5vw preferred value.

## Approach note

Matched the *result* (desktop ceiling) not the *method*: kept the fluid `clamp()`
rather than adopting tushar's stepped `text-4xl/5xl/6xl`, because home and
project-detail H1s also use `clamp()` — stepped sizing would make About the lone
exception. At >=1200px both converge to ~63.75px. Below 1024px the two differ
slightly (clamp min 2.5rem/40px vs tushar text-4xl 36px), which is acceptable and
consistent with the rest of the site.
