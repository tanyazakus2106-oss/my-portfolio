Create `src/styles/global.css` with the complete design token system. This file is the single source of truth for all visual tokens across the entire project.

The file must contain exactly:

```css
@import "tailwindcss";

/* ─── Design Tokens ─── */
@theme {
  /* Typography */
  --font-sans: "DM Sans", ui-sans-serif, system-ui, sans-serif;
  --font-mono: "JetBrains Mono", ui-monospace, monospace;

  /* Spacing scale (4px base, all multiples of 4) — per D-07 */
  --spacing-xs: 0.25rem;   /* 4px */
  --spacing-sm: 0.5rem;    /* 8px */
  --spacing-md: 1rem;      /* 16px */
  --spacing-lg: 1.5rem;    /* 24px */
  --spacing-xl: 2rem;      /* 32px */
  --spacing-2xl: 3rem;     /* 48px */
  --spacing-3xl: 4rem;     /* 64px */
  --spacing-4xl: 6rem;     /* 96px */

  /* Light mode colors (default) — per D-08 */
  --color-dominant: #FFFFFF;
  --color-secondary: #F5F5F4;
  --color-text-primary: #1C1C1C;
  --color-text-secondary: #6B7280;
  --color-border: #E5E7EB;
  --color-accent: #2563EB;
  --color-destructive: #DC2626;
}

/* Dark mode — system preference (base layer) — per D-08 */
@media (prefers-color-scheme: dark) {
  :root {
    --color-dominant: #0F0F0F;
    --color-secondary: #1A1A1A;
    --color-text-primary: #F5F5F4;
    --color-text-secondary: #9CA3AF;
    --color-border: #2D2D2D;
    --color-accent: #3B82F6;
    --color-destructive: #EF4444;
  }
}

/* Dark mode — manual toggle override (Phase 2 toggle writes .dark to <html>) — per D-08 */
:root.dark {
  --color-dominant: #0F0F0F;
  --color-secondary: #1A1A1A;
  --color-text-primary: #F5F5F4;
  --color-text-secondary: #9CA3AF;
  --color-border: #2D2D2D;
  --color-accent: #3B82F6;
  --color-destructive: #EF4444;
}
```

Key requirements:
- `@import "tailwindcss"` MUST be the first line (Tailwind v4 requires it)
- `@theme` block declares tokens into `:root` — these become usable as `var(--color-dominant)` etc.
- Dark mode colors are declared TWICE: once in `@media (prefers-color-scheme: dark)` for system preference, once in `:root.dark` for manual toggle (Phase 2). Per D-08, both layers are required from Phase 1.
- Accent color: `#2563EB` light / `#3B82F6` dark — per D-09, reserved ONLY for active nav indicator, primary CTA, link underlines on hover, focus rings, and per-project accent overrides.
- Do NOT put dark mode overrides inside `@theme` — `@theme` only writes to `:root` and cannot be scoped (per Research pitfall 3).
