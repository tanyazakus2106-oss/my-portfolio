- 5+ rows with FAIL due to the same root cause (e.g., consistent overflow under 768 due to a shared utility, or repeat touch-target failures at the same kind of element) → flag in the summary section as a candidate for a system-level fix (introduce a fluid type rule, add a body-level overflow-x guard, etc.).
